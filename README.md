## Setup

1. Clone repo
2. Run `npm install`
3. Create `.env.local` file (ask maintainer for credentials)
4. Run `npm run dev`

## Tech Stack

- Next.js 15
- Supabase (Realtime + Postgres)
- Vercel
- ESP32 (PM2.5 sensor, VOC, MQ-135, etc.)
